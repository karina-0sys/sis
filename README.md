# GitHub User Finder (Поиск пользователей GitHub)

## Описание программы

**Простое GUI**-приложение для поиска пользователей на GitHub с возможностью добавления их в избранное и сохранения списка в локальный файл

### Возможности:
- Поиск пользователей по логину
- Отображение детальной информации
- Добавление пользователей в избранное
- Сохранение избранного
- Валидация ввода

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идет в коплекте с Python)

### Запуск:
1. Откройте терминал
2. Перейдите в папку проектаъ
3. Выполните команду запуска. 
4. Просмотр информации
5. Добавление в избранное
